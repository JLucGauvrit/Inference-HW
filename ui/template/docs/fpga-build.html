FPGA Build

The FPGA flow compiles HLS kernels (MatMul, RMSNorm, Softmax, RoPE, Add, Mul) and links them into a single .xclbin. You can customize the kernels in src/ and rerun the pipeline using the Docker service.

Note: The UI currently focuses on CPU build first; the FPGA panel can be enabled next.
