Troubleshooting

  • Permission denied staging files: ensure container user matches host uid, or mount with uid/gid options and use chown -R inside the container before copying.
  • No route to host (SSH to board): verify point-to-point IPs (e.g., host 192.168.2.1/24, board 192.168.2.2/24) and that the interface is UP.
  • Exec format error on board: build with AArch64 cross-toolchain and deploy the AArch64 binary.
